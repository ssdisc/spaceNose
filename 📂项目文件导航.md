## 💻 代码类（核心实现）

### STM32嵌入式代码（src目录）
```
src/
├── main.c                    # 主程序 ⭐⭐⭐
│   ├── WiFi配置: 第112-113行
│   ├── 服务器IP: 第124行
│   └── 采样间隔: 第181行
│
├── esp8266_driver.c          # ESP8266驱动 ⭐⭐⭐
│   ├── UDP连接: ESP8266_StartConnection()
│   ├── 数据发送: ESP8266_SendUDP()
│   └── 透传模式: ESP8266_SetTransparentMode()
│
├── esp8266_driver.h          # 驱动头文件
├── stm32f4xx_hal_msp.c      # HAL底层配置
└── stm32f4xx_it.c           # 中断处理
```

**修改配置？** 只需要改 `main.c` 中的3个地方！

### 后端Python代码（backend目录）
```
backend/
├── main.py                   # FastAPI服务器 ⭐⭐⭐
│   ├── UDP服务: udp_server() - 第59行
│   ├── WebSocket: websocket_endpoint() - 第112行
│   ├── HTTP服务: root() - 第145行
│   └── 配置: UDP_PORT, UDP_HOST - 第48-49行
│
└── requirements.txt          # Python依赖列表
```

**修改端口？** 改 `main.py` 第48-49行的 `UDP_PORT` 和 `UDP_HOST`

│   │
│   └── main.js              # Vue入口文件
│

## 📌 快速命令参考

### STM32开发
```bash
# 编译
pio run

# 上传
pio run -t upload

# 串口监视器
pio device monitor
```

### 后端开发
```bash
# 安装依赖
pip install -r backend/requirements.txt

# 启动服务器
python backend/main.py

# 或者双击
start_server.bat
```

### 前端开发
```bash
# 安装依赖
cd web
npm install

# 开发模式
npm run serve

# 生产构建
npm run build
```
