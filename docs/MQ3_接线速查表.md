## 电压测试点

| 测试点 | 预期电压 | 说明 |
|:---|:---:|:---|
| JP3 VCC5 | 4.8-5.2V | 开发板 5V 输出 |
| MQ-3 VCC | 4.8-5.2V | 传感器供电 |
| MQ-3 AO | 0-5V | 传感器输出（分压前） |
| PA5 引脚 | **0-2.0V** | ADC 输入（分压后，优化） |

## ⚠️ 重要提醒

1. **必须使用分压电路！** MQ-3 输出 0-5V，STM32 ADC 只能接受 0-3.3V
2. **优化后电阻：** 15kΩ + 10kΩ，输出 2.0V，ADC 利用率 60.6%
3. **共地！** 开发板 GND 和 MQ-3 GND 必须连接在一起
4. **预热时间：** 首次使用需 24-48 小时，长期未用需 24 小时
5. **JP3 供电能力：** 最大 1000mA，MQ-3 需要 160mA，完全够用

## 快速验证步骤

```bash
1. 断电状态下连接所有线路
2. 用万用表检查：
   - JP3 VCC5 到 MQ-3 VCC 是否导通
   - 所有 GND 是否导通
   - 分压电路是否正确（15kΩ + 10kΩ）
3. 上电前再次检查！
4. 上电后测量 PA5 电压：应该在 0-2.0V 范围
5. 烧录固件，打开串口监视器（115200）
6. 观察输出：应该看到 MQ-3 状态和酒精浓度
```
