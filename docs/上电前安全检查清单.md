# ✅ 上电前安全检查清单

**在上电前，请逐项确认以下内容：**

---

## 🔌 硬件连接检查

### □ 分压电路正确
```
检查顺序（从 MQ-3 AO 开始）：
MQ-3 AO → 15kΩ → PA5 + 10kΩ → GND
         (上臂)  (中点)  (下臂)

❌ 错误：15kΩ 和 10kΩ 接反
✅ 正确：15kΩ 在上，10kΩ 在下
```

**万用表测量：**
- [ ] 15kΩ 电阻阻值：13.5kΩ - 16.5kΩ ✅
- [ ] 10kΩ 电阻阻值：9.0kΩ - 11.0kΩ ✅
- [ ] 总电阻：24kΩ - 26kΩ ✅

---

### □ 供电连接正确
```
开发板 JP3:
- VCC5 (引脚7或8) → MQ-3 VCC ✅
- GND             → MQ-3 GND ✅
```

**检查项：**
- [ ] JP3 VCC5 和 MQ-3 VCC 之间导通
- [ ] JP3 GND 和 MQ-3 GND 之间导通
- [ ] VCC 和 GND 之间**不导通**（重要！）

---

### □ 信号连接正确
```
MQ-3 AO → 分压电路 → PA5
```

**检查项：**
- [ ] MQ-3 AO 连接到 15kΩ 电阻一端
- [ ] 15kΩ 另一端连接到 PA5 和 10kΩ
- [ ] 10kΩ 另一端连接到 GND

---

### □ 无短路
**用万用表蜂鸣档检查：**
- [ ] VCC5 和 GND **不导通** ✅ 最重要！
- [ ] PA5 和 VCC5 **不导通** ✅
- [ ] PA5 和 GND **不导通**（如果导通，说明 10kΩ 短路）

## ⚡ 上电测试流程

### 第1步：空载上电
```bash
1. [ ] 仅连接 USB（不连接 MQ-3）
2. [ ] 观察 LED 是否闪烁
3. [ ] 打开串口监视器，看是否有输出
4. [ ] 用万用表测量 JP3 VCC5 电压：4.8-5.2V ✅
```

**如果第1步失败：**
- ❌ 不要继续！检查 STM32 开发板是否正常

### 第2步：连接 MQ-3（不接 AO）
```bash
1. [ ] 断电
2. [ ] 连接 MQ-3 VCC 和 GND
3. [ ] 上电
4. [ ] 用万用表测量 MQ-3 VCC：4.8-5.2V ✅
5. [ ] 用手感受 MQ-3 是否微热（预热中）
```

**如果第2步失败：**
- ❌ 检查供电连接是否正确
- ❌ 检查是否短路

### 第3步：连接分压电路（不接 PA5）
```bash
1. [ ] 断电
2. [ ] 搭建分压电路（MQ-3 AO → 15kΩ → 10kΩ → GND）
3. [ ] 上电
4. [ ] 用万用表测量分压中点电压：应 < 2.0V ✅
```

**如果第3步失败：**
- ❌ 检查电阻值是否正确
- ❌ 检查分压电路接法

### 第4步：完整连接
```bash
1. [ ] 断电
2. [ ] 连接分压中点到 PA5
3. [ ] 上电
4. [ ] 用万用表测量 PA5 电压：应 < 2.0V ✅
5. [ ] 打开串口监视器，观察输出
```

**预期串口输出：**
```
========================================

========================================

[传感器管理器] 初始化完成
[MQ-3] 初始化成功，开始预热...

[0] ===== 传感器数据 =====
  MQ-3 状态: 预热中...
  MQ-3 ADC: 1500-2500（正常范围）
  MQ-3 电压: 1.2V - 2.0V（正常范围）
  酒精浓度: 10-30 ppm（清洁空气）
============================
```

## 🚨 异常情况处理

### 情况1：PA5 电压 > 2.5V
```
原因：可能分压电路接反或电阻值错误
处理：
1. 立即断电 ❌
2. 检查分压电路
3. 用万用表测量电阻值
```

### 情况2：MQ-3 不发热
```
原因：供电未连接或 MQ-3 损坏
处理：
1. 检查 VCC 连接
2. 用万用表测量 MQ-3 VCC 引脚电压
3. 更换 MQ-3（如果损坏）
```

### 情况3：串口无输出
```
原因：固件未正确烧录或串口配置错误
处理：
1. 检查串口波特率（115200）
2. 检查串口号是否正确
3. 重新烧录固件
```

### 情况4：ADC 读数为 0 或 4095
```
原因：PA5 未连接或短路
处理：
1. 断电
2. 用万用表检查 PA5 连接
3. 检查是否有短路
```

## 📋 应急停机条件

**立即断电，如果：**
- ⚠️ STM32 或 MQ-3 异常发热
- ⚠️ 闻到烧焦味
- ⚠️ PA5 电压 > 3.0V
- ⚠️ 电流异常增大
- ⚠️ 出现异常噪音或火花
