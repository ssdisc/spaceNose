# 🔄 分压电路优化说明

## 修改概要

**从：** 10kΩ + 20kΩ → **改为：** 10kΩ + 15kΩ

---

## 📊 参数对比

| 参数 | 原方案 (10k+20k) | 优化方案 (10k+15k) | 改善 |
|:---|:---:|:---:|:---:|
| **输出电压** | 1.67V | **2.0V** | ↑ 19.8% |
| **ADC利用率** | 50.6% | **60.6%** | ↑ 10% |
| **实际分辨率** | ~2070级 | **~2482级** | ↑ 412级 |
| **安全裕量** | 1.63V | **1.3V** | ↓ 0.33V |
| **安全系数** | 1.98x | **1.65x** | 仍然安全 ✅ |

---

## 🎯 优化理由

### 1. 提高测量精度
```
分辨率提升：412 个 ADC 级别
相当于精度提高了 19.8%
```

### 2. 更好的 ADC 利用率
```
从 50.6% 提升到 60.6%
减少了信号浪费
```

### 3. 仍然保持安全
```
安全裕量：1.3V
即使 MQ-3 输出 5V，PA5 只会收到 2.0V
远低于 3.3V 的最大值
```

---

## 🔧 代码修改

### sensor_manager.c

**修改前：**
```c
#define VOLTAGE_DIVIDER_RATIO   1.5f    // 5V → 3.3V
```

**修改后：**
```c
#define VOLTAGE_DIVIDER_RATIO   2.5f    // 5V → 2.0V (15kΩ + 10kΩ)
```

**计算方法：**
```
分压输出 = 5V × 10kΩ / (15kΩ + 10kΩ) = 2.0V
分压比 = 原电压 / 分压后电压 = 5V / 2.0V = 2.5
```
